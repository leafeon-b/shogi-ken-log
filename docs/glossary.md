# 用語集

## ドメイン用語

| 日本語   | 英語(システム内部用) | 説明                                                     |
| -------- | -------------------- | -------------------------------------------------------- |
| 研究会   | Circle               | ある程度決まったメンバーで継続的に開催される将棋の対局会 |
| 開催回   | CircleSession        | 研究会に属する 1 回分の開催                              |
| 管理者   | Admin                | 研究会・開催回を管理できる参加者                         |
| メンバー | Member               | 研究会に所属して対局する参加者                           |

## システム用語（実装上の概念）

| 日本語             | 英語(システム内部用) | 説明                                                                           |
| ------------------ | -------------------- | ------------------------------------------------------------------------------ |
| 論理削除           | Soft Delete          | 物理削除せず、削除フラグ（例: deletedAt）で「削除済み」を表す                  |
| 対局結果の編集履歴 | MatchHistory         | 対局結果の作成・修正・削除の履歴。編集者・編集日時・スナップショットを保持する |
| 開催回内の対局順序 | Match Order          | 開催回内での対局結果の並び順。開催回ごとに一意で、論理削除後も再利用しない     |
